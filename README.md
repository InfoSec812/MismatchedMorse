# Programming Interview Question - Morse Code Mismatches

Problem description is in [Morse Mismatches](Morse Mismatches.doc?raw=true)

## Prerequisites
* Java 8
* Maven >=3.0.0

## Building

```
mvn clean package
```

## Running Tests

```
mvn test
```

## Executing

```
mvn package
java -jar target/morse-mismatches-1.0-SNAPSHOT.jar sample.txt
```
